# Discover Italy

Discover Italy — веб-приложение, разработанное на основе фреймворка Bottle. Проект представляет собой тематический сайт, посвящённый культуре Италии, её городам, достопримечательностям и национальной кухне.

---

## Начало работы

Данные инструкции позволят получить копию проекта и запустить его на локальной машине для разработки и тестирования. Информация о возможном развертывании на сервере представлена в разделе «Развертывание».

---

## Необходимые условия

Для работы с проектом необходимо установить программное обеспечение и выполнить его настройку:

- Python 3.9 или выше  
- pip  
- Git  
- Среду разработки (Visual Studio или VS Code)

Пример проверки версии Python:

```bash
python --version
```

Пример проверки pip:

```bash
pip --version
```

## Установка

Ниже приведена последовательность действий для запуска среды разработки.

### Шаг 1. Клонирование репозитория

```bash
git clone <ссылка_на_репозиторий>
cd Discover_Italy
```

### Шаг 2. Создание виртуального окружения

```bash
python -m venv venv
```

Активация (Windows):

```bash
venv\Scripts\activate
```

### Шаг 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### Шаг 4. Запуск приложения

```bash
python app.py
```

После запуска открыть в браузере:

```
http://localhost:8080
```

Пример демонстрации: переход по маршрутам `/`, `/cities`, `/kitchen`, `/contact` для просмотра страниц сайта.

## Запуск тестов

В проекте отсутствуют автоматизированные тесты, так как работа носит учебный характер.

### Сквозное тестирование

Проверяется:

- корректный запуск сервера;
- отсутствие ошибок в консоли;
- корректная работа маршрутов;
- отображение статических файлов.

Пример: открыть страницу `/cities` и убедиться в корректном отображении контента.

### Проверка стиля кода

Проверяется:

- соблюдение структуры Bottle-проекта;
- корректное разделение логики и шаблонов;
- правильное подключение CSS и статических ресурсов.

Пример: проверка корректности работы layout.tpl и маршрутов в routes.py.

---

## Развертывание

Для развертывания проекта на сервере необходимо:

1. Установить Python.
2. Установить зависимости из файла requirements.txt.
3. Использовать WSGI-сервер (например, gunicorn).
4. Настроить доменное имя и серверное окружение.

В рамках лабораторной работы проект запускался локально.
